## Create a Cashback
#### [Let's check how to add cashbackCopied!](https://docs.tryspeed.com/docs/send-payments/cashback/create-cashback#let's-check-how-to-add-cashback)
You can create a cashback by filling out the details, such as name, description, type, amount, start date, end date, and expiry in days, in the form available on the Speed web application.
Log in to your account and follow the steps below to create a cashback.
  1. On the left sidebar, click [Send Payments > Cashback](https://app.tryspeed.com/cashbacks).
  2. Click Create New.
  3. After clicking the Create New button, you are redirected to the Create page. Fill out the required details, like name, type, amount, and start date.
  4. The end date refers to when you intend to conclude this offer. If left blank, the cashback will remain valid indefinitely.
  5. The expiration period in days indicates the duration within which you can redex the offer after claiming it.
  6. Click the Create Cashback button. This will redirect you to the detail page, where you can see the cashback you created and its details.
  7. Now, associate it with different speed resources and send rewards.


#### [Associate a CashbackCopied!](https://docs.tryspeed.com/docs/send-payments/cashback/create-cashback#associate-a-cashback)
  * Web App
  * API


While creating any of the Speed receive payments resources through the web application, you can associate a single cashback.
  * Checkout link
  * Payment link


For this, log in to your account and follow the steps below:
  1. On the left sidebar, click Receive Payments > Checkout/Payment Links.
  2. Click Create New.
  3. After clicking the Create New button, you are redirected to the Create page. Fill out the required details.
  4. In this form, you will find the “Add Cashback” checkbox under the options section. Tick the checkbox and select a single cashback from the list of available cashbacks.
  5. If there is no existing cashback and you wish to establish a new one, you can follow the steps outlined [above](https://docs.tryspeed.com/docs/send-payments/cashback/create-cashback#let's-check-how-to-add-cashback) to create it and subsequently link it.


> **Note -** One payment link/ checkout link can have only one cashback associated at a time, but one cashback can be associated with multiple payment links/ checkout links at a time.
