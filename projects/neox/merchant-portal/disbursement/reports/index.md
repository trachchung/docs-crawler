## 
1. Disbursement history:
List of discursement history
  * Display details of disbursement transactions
  * Search by optional information: Reference code, Request code, Transaction code, Status, Request Date.
  * Export transaction report with optional information: Requested Date, Status.


## 
2. Disbursement report:
  * Display detailed information of the reconciliation period
  * Search by optional information: Reconciliation Code, Reconciliation period
  * Export transaction report with optional information: Reconciliation period.
