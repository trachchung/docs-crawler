NeoX COL is a system that provides an interactive interface that allows Merchants to perform collection operations via APIs.
Merchants can perform collection and manage transactions directly on Merchant Portal or using APIs in this document.
The merchant must register an account on Merchant Portal and activate the Collection Service before using it.
