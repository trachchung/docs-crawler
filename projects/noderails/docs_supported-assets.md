# Supported Chains and Tokens

Browse all chains and tokens currently enabled in NodeRails.

✅

Need another chain?

We can add any chain within 24 hours. Send us feedback or a feature request on the landing page, or reach out on Telegram.

💡

Quick integration tip

In API and SDK requests, pass `allowedChains` using chain IDs (for example `8453`) and `allowedTokens` using token symbols or token keys (for example `USDC` or `USDC-8453`). See the [Quick Start guide](https://www.noderails.com/docs/getting-started) for usage examples.
