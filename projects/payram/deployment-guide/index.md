## 
PayRam installation methods
You can set up your PayRam server using one of two deployment methods, each designed to suit different experience levels and setup preferences.
⚡ **Quick Setup**
Use the PayRam script to install, configure, and start accepting payments in under 10 minutes!
⚙️ **Advanced Setup**
Install PayRam on Docker by manually configuring containers, storage, networks, and variables.
