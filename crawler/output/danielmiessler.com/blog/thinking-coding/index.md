# Coding is Thinking
We can't stop learning fundamentals just because tech can do them
March 16, 2025
[ #future](https://danielmiessler.com/archives/?tag=future)[ #technology](https://danielmiessler.com/archives/?tag=technology)[ #ai](https://danielmiessler.com/archives/?tag=ai)[ #top](https://danielmiessler.com/archives/?tag=top)
 Self-attending…
Not learning to code just because there are AI coding agents is like not learning how to think because there are talk shows.
  * Writing = thinking.
  * Creating = imagining.
  * Coding = building.


If you're in tech in 2025 and you can't do these things, your career is at risk.
You have to adapt or you will be replaced.
Thinkers and creators will be able to build/execute faster than executors will be able to think and create.
Become a thinker and/or creator. And if you're not sure how to start, start by reading.
2-4 good reading per month.
